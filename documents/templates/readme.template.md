## 섹션 가이드라인

| 섹션 | 필수 여부 | 라인 수 | 목적 |
| ------- | -------- | ----- | ------- |
| Title + Badges | ✅ | 1-2 | 프로젝트 이름, 상태 배지 |
| One-Liner | ✅ | 1 | 프로젝트를 가장 간단하게 설명 |
| Quick Start | ✅ | 10-15 | 설치 + 최소한의 작동 예제 |
| Features | ✅ | 3-7 | 핵심 기능, 너무 길지 않게 작성 |
| Installation | ✅ | 10-20 | 사전 요구 사항, 단계, 검증 |
| Basic Usage | ✅ | 5-10 | 코드가 포함된 가장 일반적인 사용 사례 |
| Documentation Links | ✅ | 5-8 | 다른 문서로의 포인터 |
| Contributing | ⚪ | 3-5 | 이슈 보고 방법, PR 제출 방법 |
| License | ✅ | 1 | 링크가 포함된 라이선스 유형 |
